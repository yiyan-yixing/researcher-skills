# 论文深度阅读检查清单

> 每次读论文时对照此清单，确保不遗漏关键信息。

## 第一遍：结构扫描（2min）

- [ ] 标题和摘要 → 写下方法预测和结果预测
- [ ] 作者和机构 → 他们之前做过什么？有什么偏见？
- [ ] 引用网络 → 他们引用了谁？没引用谁？
- [ ] 图表标题 → 不看图，只看标题，猜图的内容

## 第二遍：限制和附录优先（10min）

- [ ] Limitations 章节 → 完整阅读
  - 他们承认了什么？
  - 他们没承认但你看到的限制？
- [ ] Appendix → 扫描每个小节
  - 有没有失败的消融实验？
  - 有没有"我们尝试了 X 但不工作"的脚注？
  - 超参数表格在哪里？是否完整？
  - 数据集细节是否充分？

## 第三遍：方法深挖（10min）

- [ ] 方法的每个步骤
  - 这个步骤的必要条件是什么？如果去掉会怎样？
  - 他们是否对比了简化版本？
  - 计算成本是多少？
- [ ] 关键假设
  - 方法在什么条件下会失效？
  - 他们测试了这些条件吗？

## 第四遍：结果审视（5min）

- [ ] 数字和基线对比
  - 基线是否经过调优？（调基线到痛处）
  - 误差棒是否合理？标准差 vs 标准误？
  - 统计显著性是否报告？
- [ ] 和 Step 1 预测对照
  - 偏差方向：乐观了还是悲观了？
  - 偏差原因：遗漏了什么信息？

## 第五遍：机会识别（3min）

- [ ] 他们没做的实验 → 那是你的机会
- [ ] 他们回避的基线 → 可能那个基线很强
- [ ] 他们没讨论的限制 → 值得深挖
- [ ] 跨领域迁移潜力 → 这个方法在其他领域适用吗？
